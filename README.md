# homebrew-cask

### search

```sh
$ brew search --cask $target_name
```

### instal / reinstall

```sh
$ brew install --cask $target_name

$ brew reinstall --cask $target_name
```

### uninstall

```sh
$ brew uninstall --cask $target_name
```

### dump Brewfile

```sh
# output Brewfile in current directory  
$ brew bundle dump
```

### install with Brewfile

```sh
$ brew bundle
```

### uninstall with Brewfile

```sh
$ brew bundle cleanup
```
